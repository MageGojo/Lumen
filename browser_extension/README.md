# Lumen 下载接管(IDM 同款)

一个 MV3 Chrome 扩展,**接管浏览器的下载**——你在网页上点任何下载,它会拦下来,自动交给 Lumen 桌面应用多线程下载(就像 IDM 的集成模块)。另外可选嗅探页面里的视频 / 媒体。无需上架商店,开发者模式直接加载文件夹。

## 加载(开发者模式)

1. 打开 Lumen 应用(它会在本地启动 `http://127.0.0.1:8787` 接收桥)。
2. Chrome 打开 `chrome://extensions`。
3. 右上角打开「开发者模式」。
4. 点「**加载已解压的扩展程序**」(不要用"打包扩展程序"),选择本 `browser_extension` 目录。
5. 工具栏出现「Lumen 下载接管」即成功。

> 不要"打包"成 `.crx`:Chrome 已禁止安装商店外的 `.crx`。非商店扩展只能"加载已解压"。
> Edge / Brave 同样支持,且对非商店扩展更宽松。

## 核心:接管下载

- 在网页点下载链接 / 下载按钮,扩展会在下载刚开始时**取消浏览器下载**,把地址(带 `Referer` 与文件名)发给应用,由应用多线程下载并自动前置窗口。
- `.torrent` 下载同样被接管 → 应用自动走 aria2(BitTorrent)。
- **安全网**:若应用未运行 / 不可达,扩展不会拦截,交回浏览器正常下载,绝不会"吞掉"文件。
- 弹窗顶部开关可随时「接管中 / 已关闭」;关闭后浏览器恢复原生下载。

## 可选:嗅探页面媒体

- 同时监听页面的视频 / 音频请求与 `<video>` 源,识别 HLS(`.m3u8`)/ DASH(`.mpd`)/ 直链视频,弹窗「本页媒体」里列出,可「发送 / 复制」。
- 可在设置里关掉嗅探,做成纯粹的下载接管器。

## 右键菜单

- 链接 / 视频 / 图片上右键 →「用 Lumen 下载此链接」。
- 页面空白处右键 →「用 Lumen 解析此页面」(短视频平台走应用内 Apizero 解析)。

## 设置(弹窗底部「设置」)

- **接管浏览器下载**:核心开关。
- **嗅探页面媒体**:可选。
- **本地桥端口**:默认 8787,需与应用一致。
- **最小接管体积(MB)**:小于该值交给浏览器;0 = 全部接管。
- **不接管的后缀**:逗号分隔,默认 `m3u8,mpd`(流清单交浏览器/页面)。

## 下载路由(应用侧)

- 带 `Referer` 的接管下载 → aria2(携带 Referer + 浏览器 UA,绕过防盗链 403)。
- 磁力 / `.torrent` → aria2(BitTorrent)。
- 普通直链 → Surge(HTTP 多线程)。
- 短视频平台「页面」→ Apizero 解析多清晰度。

## 已知限制

- 需应用处于运行状态(弹窗顶部状态点为绿色=已连接);未运行时接管自动让位给浏览器。
- HLS/DASH 清单只嗅探展示,合并为单文件需 ffmpeg(后续接入)。
- 开发者模式加载的扩展,Chrome 在更新 / 重启后可能临时禁用,回 `chrome://extensions` 打开即可。
